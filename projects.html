Internal Tool – Arena PLM (PTC)

AI Release Notes & Docs Assistant

Problem: Release notes and help updates were time-consuming and relied on tribal knowledge.

Role: Designed an AI program that reads code diffs and Jira tickets, drafts release notes, maps changes to help topics, and surfaces documentation gaps; owned prompts, gold data, and eval loops.

Solution: Multi-step pipeline: ingest diffs and tickets → classify changes → draft user-friendly notes → map to help topics and suggest edits.

Impact: Reduced manual drafting time, improved alignment between product, docs, and support, and created a reusable pattern for AI automation.
